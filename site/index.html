A modular toolkit that transforms Claude Code into a personal knowledge system, journal, task manager, and more.

New to Claude Code? Start here

What's inside

Four layers. Install what you need.

◆

Core

The foundation. Session hooks, git sync, write guards, a specs system for documenting features, persistent memory, and commands for updates and contributions.

★

Life

A personal knowledge system. Conversational journaling, a living encyclopedia that Claude maintains about you, and automatic Google Drive backup.

⚡

Productivity

Task management and communication. Process your Todoist inbox, build custom skills, and read or send text messages through Claude.

❖

Modules

Optional add-ons for specific domains. Currently includes Arizona elections tracking and fiscal impact analysis tools.

Each layer is independent. Life and Productivity both build on Core, but don't depend on each other. Want the technical details?

Get started

Two commands. That's it.

$ curl -fsSL https://raw.githubusercontent.com/itsdestin/claudifest-destiny/master/bootstrap/install.sh | bash
> powershell -ExecutionPolicy Bypass -c "iwr -useb https://raw.githubusercontent.com/itsdestin/claudifest-destiny/master/bootstrap/install.ps1 -OutFile install.ps1; .\install.ps1"

Then open Claude Code in your terminal and say "set me up"

A terminal is a program where you type commands instead of clicking buttons. On Mac, press Cmd+Space, type "Terminal", and press Enter. On Linux, press Ctrl+Alt+T.
PowerShell is Windows' built-in command-line tool. Press the Windows key, type "PowerShell", and press Enter. A blue window will appear.
