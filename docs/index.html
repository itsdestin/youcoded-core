Built for my friends. Shared with everyone.

Claude can create every type of file, search the web, open any app, and navigate your screen if you desire. It is designed to be a complete personal assistant.

It will not do any of this without your permission.

Get started

Step 1: Run the installer

Open Terminal: Press Cmd + Space to open Spotlight, type Terminal, and press Enter. A window with a blinking cursor will appear.

Paste this and press Enter:

$ curl -fsSL https://raw.githubusercontent.com/itsdestin/claudifest-destiny/master/bootstrap/install.sh | bash

Open Windows Terminal: Press the Windows key, type Terminal, and press Enter. A window with a blinking cursor will appear.

Paste this and press Enter:

> powershell -ExecutionPolicy Bypass -c "iwr -useb https://raw.githubusercontent.com/itsdestin/claudifest-destiny/master/bootstrap/install.ps1 -OutFile install.ps1; .\install.ps1"

Step 2: Talk to Claude

In that same terminal window, type:

$ claude

and press Enter. Claude will open. Then type:

> Set me up.

Claude will walk you through the rest — choosing what to install, setting up your preferences, and making sure everything works.

WARNING: You may see lots of red "error" text and be asked for a multitude of confirmations while Claude completes this process. Don't worry; this is to be expected. Trust Claude (he is all-knowing) and approve any prompts.

While you wait for Claude to complete setup, read this.

If you'd like to know more, review the additional documentation at the bottom of this page.

◆

Core

The foundation. Session hooks, git sync, write guards, a specs system for documenting features, persistent memory, and commands for updates and contributions.

★

Life

A personal knowledge system. Conversational journaling, a living encyclopedia that Claude maintains about you, and automatic Google Drive backup.

⚡

Productivity

Task management and communication. Process your Todoist inbox, build custom skills, and read or send text messages through Claude.

❖

Modules

Optional add-ons for specific domains. Currently includes Arizona elections tracking and fiscal impact analysis tools.

Each layer is independent. Life and Productivity both build on Core, but don't depend on each other. Want the technical details?
