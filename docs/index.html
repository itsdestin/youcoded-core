DestinCode

Claude, but yours.

A more personal, more customizable Claude — with themes, a skill marketplace, multiplayer mini-games, and remote access from any browser. Runs on Windows, macOS, Linux, and Android. Powered by Claude Code. Open source.

What is this?

More than a chat window.

DestinCode is a full app for Claude Code — Anthropic's AI agent that can create and edit files, search the web, run terminal commands, and navigate your screen. Think of it as the desktop and mobile home Claude Code deserves — with a chat UI you can theme, a marketplace full of downloadable themes, skills, games, and integrations, multiplayer mini-games for the downtime, and remote web access from any browser on your network.

The optional DestinClaude toolkit add-on takes it further. Journal about your day and it pulls out tasks, calendar events, and details worth remembering — no templates, no forms. Over time, Claude builds a searchable personal encyclopedia: who you met, what you discussed, the context you always forget. Snap a screenshot of a flyer, and the event lands on your calendar. Mention a deadline in passing, and it shows up in your task list. You just talk. The structure happens on its own.

Nothing happens without your permission. DestinCode will always ask before taking any action.

Your pocket

Full Claude Code. On your phone.

This isn't a watered-down mobile companion. DestinCode on Android runs the real Claude Code binary natively on your device — same skills, same plugins, same power as desktop.

Everything you set up on your computer works on your phone too. Themes, marketplace skills, hooks, slash commands — all of it. Start a project on your laptop, pick it up on Android during your commute, then continue at your desk. Your setup follows you.

Full environment A real Linux shell runs on-device — bash, Node.js, Python, git, and more, installed and managed automatically on first launch.
Native + familiar Native Android for file pickers, QR scanning, and device integration. The chat UI is the same interface you use on desktop.
Your plugins travel Every skill, theme, hook, and MCP server you install on desktop works on Android too. No separate setup needed.
iPhone? An iOS app isn't currently possible — Claude Code requires a native Linux environment that Apple doesn't allow on iPhone. But you can use DestinCode from your iPhone's browser by connecting to any computer running the app via remote access. Same full UI, just over your network.

Common questions

FAQ

Claude.ai is a chat website. DestinCode is a full app built on Claude Code — a different, more powerful mode of Claude that can create files, run terminal commands, manage your computer, and interact with external services. Think of claude.ai as texting Claude, and DestinCode as giving Claude hands. On top of that, DestinCode adds custom themes, a marketplace full of downloadable skills, games, and integrations, multiplayer mini-games, remote access, and (with the optional toolkit) journaling, a personal encyclopedia, and messaging integrations that claude.ai doesn't have. The goal is to make AI feel fun, social, and personal — not like another boring productivity tool.
Everything DestinCode and its toolkit create — journal entries, your Encyclopedia, tasks — is stored in your own Google Drive or iCloud account. Nothing is sent to third-party servers beyond the services you explicitly connect. Your data stays yours.
The $20 goes to Anthropic for a Claude Pro subscription, which gives you access to Claude Code — the AI that powers everything. DestinCode itself is free and open source. You're paying for the AI, not the app.
Windows, macOS, Linux, and Android are all fully supported with native apps. You can also access DestinCode remotely from any web browser. Apple ecosystem integrations (iMessage, Apple Notes, etc.) are only available on macOS.
Not at all. DestinCode was built entirely by a non-developer using Claude Code itself. The app is designed for everyone — students, professionals, anyone who uses AI regularly. If you can send a text message, you can use DestinCode.
Yes — just describe the vibe you want and Claude builds it. Use /theme-builder inside the app. Themes support wallpapers, glassmorphism, particle effects, mascots, custom icons, and more. You can share your themes with friends or publish them to the community marketplace.
DestinCode always asks for permission before taking actions, so it's hard to break things accidentally. If something does go wrong, the system uses git version control under the hood — you can always roll back.
